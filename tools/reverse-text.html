About the Transformation Modes

  • Reverse String: Reverses every character in the text. "Hello World" → "dlroW olleH".
  • Reverse Words: Reverses the order of words. "Hello World" → "World Hello".
  • Reverse Letters: Reverses the letters inside each word. "Hello World" → "olleH dlroW".
  • Reverse Lines: Reverses the order of lines in a multi-line text.
  • Flip (Upside Down): Maps each character to its upside-down equivalent. "Hello" → "oןןǝH".

Frequently Asked Questions (FAQ)

How does 'Flip (Upside Down)' work?

This mode uses a character map to replace each letter or symbol with a Unicode character that looks like its upside-down version. Not all characters have a flipped equivalent, so some may remain unchanged.

Does this tool work with emojis and special characters?

Yes. Modern JavaScript is aware of multi-byte characters like emojis (e.g., 👍), so reversing a string like "Hi 👍" will correctly result in "👍 iH" instead of breaking the emoji.
